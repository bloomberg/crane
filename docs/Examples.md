# Examples

A curated selection of C++ files generated by Crane (accompanied with their source Rocq programs) is available to view in the [`tests`](https://github.com/bloomberg/crane/tree/main/tests) directory. The tests are organized thematically, with [`tests/basics`](https://github.com/bloomberg/crane/tree/main/tests/basics) containing traditional, functional Rocq programs, and [`tests/monadic`](https://github.com/bloomberg/crane/tree/main/tests/monadic) containing effectful examples.
