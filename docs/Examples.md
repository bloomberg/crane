# Examples

A curated selection of C++ files generated by Crane (accompanied with their source Rocq programs) is available to view in the [`tests`](https://github.com/bloomberg/crane/tree/main/tests) directory. The tests are organized thematically:

- [`tests/basics`](https://github.com/bloomberg/crane/tree/main/tests/basics) — Traditional functional Rocq programs (lists, trees, modules, type classes, etc.)
- [`tests/monadic`](https://github.com/bloomberg/crane/tree/main/tests/monadic) — Effectful examples (IO, STM, threads, interaction trees)
- [`tests/regression`](https://github.com/bloomberg/crane/tree/main/tests/regression) — Regression tests for previously fixed issues
- [`tests/wip`](https://github.com/bloomberg/crane/tree/main/tests/wip) — Work-in-progress tests for known issues and experimental features
